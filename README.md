# Simple Notes App

Full-stack (React + Node.js + Express + MongoDB).

Project structure:

- `backend/` — Express API, Mongoose models
- `frontend/` — Vite + React frontend

Features:
- Create, read, update, delete notes
- MongoDB persistence
- Simple React UI

1) Backend

Server runs on `http://localhost:5000` by default.

2) Frontend

Frontend runs on `http://localhost:3000` by default. It expects the API at `http://localhost:5000/api`.

Important Note: Node Modules are not installed, you have to install it on your own local computer with 'npm install' cmd.
